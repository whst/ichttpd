说明：

1、后端使用 Python 3.4.1 开发；仅使用了 Python 标准模块，以下是使用到的模块列表： string、cgi、sqlite3 、datetime、time、os；

2、理论上可以运行于所有支持 Python 3 的平台，包括 Windows、Linux、OS X、BSD等。

组内测试环境为 openSuSE 13.2，故请评委使用 CentOS 7 测试；

3、文档均为 UTF-8 编码。


运行：

解压，在解压得到的 wishboard 内运行“python3 simple_httpd.py”；

该 simple_httpd.py 脚本是我们测试时用的简易服务器。

访问地址为http://localhost:8080/，若默认端口8080被占用，请修改该文件中的 port 变量至一个可用的值。
